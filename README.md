## ğŸ¨ Customization Guide

### Change Name & Message

Open `Birthday_Card.html` and look for text like:

```html
<h1>Happy Birthday!</h1>

```

Replace it with your own content.

---

### Change Colors

Find the CSS section (inside `<style>`):

```css
body {
  background-color: #ffcccb;
  color: #333;
}
```

Modify colors using:

* HEX (`#ff5733`)
* RGB (`rgb(255, 87, 51)`)
* Named colors (`blue`, `pink`)
